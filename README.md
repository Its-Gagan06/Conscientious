# 🌍 Community Reporter

Community Reporter is a modern web application that empowers citizens to report civic issues in their local communities. Users can easily submit reports for problems such as potholes, waste management, broken streetlights, water leakage, and other public infrastructure issues, helping authorities respond more efficiently.

## ✨ Features

* 📝 Report community issues with detailed descriptions
* 📸 Upload photo evidence for better issue verification
* 📍 Attach the current location using GPS
* 🏷️ Categorize issues (e.g., Waste Management, Potholes, Water Leakage, Streetlights)
* 🚀 Clean, responsive, and user-friendly interface
* ✅ Instant confirmation after successful report submission
* 🎨 Modern UI with smooth animations

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/<your-repository>.git
```

### 2. Navigate to the project folder

```bash
cd <your-repository>
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```
